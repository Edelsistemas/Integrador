# Módulo: SapItemService

## Propósito

Encapsular operaciones contra SAP B1 Service Layer para el maestro de artículos.

## Responsabilidades

- Login indirecto por `CommonAPIService`.
- Consultar existencia de `ItemCode`.
- Leer datos actuales mínimos para comparar inactivación.
- Crear o actualizar artículos.
