# Reviewer Agent

## Propósito

Pendiente de ajustar por tarea concreta.

## Cuándo se usa

Cuando una tarea encaja con este rol.

## Entradas

- Wiki.
- Spec.
- Código.
- Evidencia.

## Salidas

- Cambios documentados.
- Evidencia.
- Handoff si corresponde.

## Límites

No modificar producción ni secretos. Requiere revisión humana para cambios sensibles.

## Checklist

- Leí spec.
- Revisé riesgos.
- Ejecuté harness aplicable.
- Actualicé documentación.
