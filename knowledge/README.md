# Base de conocimiento - Integrador Teamcenter a SAP B1

## Propósito

Esta carpeta contiene la documentación estructurada del sistema conocido como **Integrador**, una aplicación productiva legacy que sincroniza artículos creados/aprobados en Siemens Teamcenter PLM hacia SAP Business One mediante Service Layer, usando una base intermedia SQL Server.

El objetivo es que una persona técnica pueda continuar el análisis y desarrollo desde VSCode/Codex sin depender del chat original.

## Principios de uso

1. El código fuente en ejecución es la última verdad técnica.
2. Las planillas de mapeo representan intención funcional y acuerdos de negocio.
3. Los correos sirven como evidencia histórica de decisiones, pruebas y correcciones.
4. Toda nueva modificación funcional debe partir de una spec aprobada.
5. No se deben exponer secretos reales en esta wiki.
